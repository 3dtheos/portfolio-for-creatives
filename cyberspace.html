Cyberspace

With motion graphics and 3D animation becoming more and more common in graphic design work, IK decided to create a font that would combine the two.

Cyberspace is a font catered towards motion graphics artists like myself who desire a little more control over each glyph in a typeface when doing 3D work in Blender.

Cyberspace includes not just a font file, but also a 3D file containing 3d meshes with rigs, which allow extra control over each letter, opening up new opportunities for typographic animation and non-destructive editing.
